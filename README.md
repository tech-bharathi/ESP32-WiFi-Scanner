# ESP32 WiFi Scanner

A project using the ESP32 board to scan nearby WiFi networks 
and display details via Serial Monitor.

## Features
- Network Name (SSID)
- Signal Strength (RSSI)
- Channel Number
- MAC Address
- Encryption Type (WPA2, WPA3, etc.)

## Requirements
- ESP32 Board
- Arduino IDE
- WiFi.h Library (built-in)

## Usage
1. Upload the code to your ESP32 board
2. Open Serial Monitor in Arduino IDE
3. Set baud rate to **115200**
4. View scanned WiFi networks instantly
